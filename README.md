## Overview
The bonus assignment focuses on developing a simple code generator that can automatically translate models into program code. The project aims to propose an algorithm to translate models described in your syntax to Java or Python code. Then, it implement a proof-of-concept solution through Python scripts. Then, a reflection on the code generator created.
